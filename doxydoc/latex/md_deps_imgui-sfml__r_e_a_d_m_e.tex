If you have S\+F\+ML installed on your system, you don\textquotesingle{}t need to set S\+F\+M\+L\+\_\+\+D\+IR during configuration.

You can also specify {\ttfamily B\+U\+I\+L\+D\+\_\+\+S\+H\+A\+R\+E\+D\+\_\+\+L\+I\+BS=ON} to build Im\+Gui-\/\+S\+F\+ML as a shared library. To build Im\+Gui-\/\+S\+F\+ML examples, set {\ttfamily I\+M\+G\+U\+I\+\_\+\+S\+F\+M\+L\+\_\+\+B\+U\+I\+L\+D\+\_\+\+E\+X\+A\+M\+P\+L\+ES=ON}.

After the building, you can install the library on your system by running\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cmake -\/-\/build . -\/-\/target install}
\end{DoxyCode}


If you set {\ttfamily C\+M\+A\+K\+E\+\_\+\+I\+N\+S\+T\+A\+L\+L\+\_\+\+P\+R\+E\+F\+IX} during configuration, you can install Im\+Gui-\/\+S\+F\+ML locally.

\DoxyCodeLine{target\_link\_libraries(my\_target PRIVATE ImGui-\/SFML::ImGui-\/SFML)}
